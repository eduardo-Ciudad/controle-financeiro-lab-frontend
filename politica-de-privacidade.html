Política de Privacidade

Última atualização: 17/07/2026

Esta Política de Privacidade descreve como o Controle Financeiro ("nós", "nosso") coleta, utiliza, armazena e protege os dados pessoais dos usuários ("você", "seu"), em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD — Lei nº 13.709/2018).

1. Dados que Coletamos

1.1. Dados fornecidos diretamente pelo usuário:

  • Nome completo
  • Endereço de e-mail
  • Senha (armazenada de forma criptografada com BCrypt)
  • Dados financeiros inseridos voluntariamente (receitas, despesas, categorias)

1.2. Dados coletados automaticamente:

  • Endereço IP
  • Tipo de navegador e sistema operacional
  • Data e hora de acesso
  • Páginas visitadas dentro da Plataforma
  • Cookies (conforme detalhado na Política de Cookies)

2. Como Utilizamos Seus Dados

Utilizamos seus dados para:

  • Criar e gerenciar sua conta na Plataforma;
  • Fornecer, manter e melhorar os serviços oferecidos;
  • Processar e exibir suas movimentações financeiras;
  • Enviar comunicações essenciais sobre sua conta (verificação de e-mail, recuperação de senha);
  • Garantir a segurança e prevenir fraudes;
  • Cumprir obrigações legais e regulatórias.

3. Base Legal para o Tratamento (LGPD Art. 7º)

O tratamento dos seus dados pessoais é fundamentado nas seguintes bases legais:

  • Execução de contrato (Art. 7º, V): necessário para a prestação do serviço contratado;
  • Consentimento (Art. 7º, I): quando aplicável, especialmente para cookies não essenciais;
  • Legítimo interesse (Art. 7º, IX): para melhorias na Plataforma e segurança;
  • Cumprimento de obrigação legal (Art. 7º, II): quando exigido por lei.

4. Compartilhamento de Dados

4.1. Não vendemos, alugamos ou comercializamos seus dados pessoais.

4.2. Seus dados podem ser compartilhados com:

  • Provedores de infraestrutura e hospedagem (servidores, banco de dados) — estritamente para operação da Plataforma;
  • Autoridades legais, quando exigido por lei ou decisão judicial.

5. Armazenamento e Segurança

5.1. Os dados são armazenados em servidores com as seguintes medidas de segurança:

  • Criptografia de senhas com BCrypt;
  • Autenticação via tokens JWT com expiração controlada;
  • Comunicação exclusivamente via HTTPS (SSL/TLS);
  • Rate limiting para prevenção de ataques de força bruta;
  • Isolamento de dados em ambiente multi-tenant;
  • Backups regulares do banco de dados.

5.2. Apesar de empregarmos medidas razoáveis de segurança, nenhum sistema é 100% seguro. Recomendamos que o usuário adote boas práticas como senhas fortes e únicas.

6. Seus Direitos (LGPD Art. 18)

Como titular dos dados, você tem direito a:

  • Confirmação da existência de tratamento de dados;
  • Acesso aos dados pessoais tratados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos;
  • Portabilidade dos dados;
  • Eliminação dos dados tratados com base no consentimento;
  • Informação sobre com quem seus dados foram compartilhados;
  • Revogação do consentimento.

Para exercer seus direitos, entre em contato pelo e-mail: eduardociudadfigueredo@gmail.com

7. Retenção de Dados

7.1. Os dados pessoais são mantidos enquanto a conta do usuário estiver ativa.

7.2. Após o encerramento da conta, os dados serão eliminados em até 30 dias, exceto quando houver obrigação legal de retenção.

7.3. Dados financeiros anonimizados podem ser retidos para fins estatísticos.

8. Transferência Internacional

8.1. Seus dados podem ser processados em servidores localizados fora do Brasil (provedores de cloud como Hostinger, Vercel).

8.2. Nesses casos, garantimos que os provedores adotem medidas de proteção compatíveis com a LGPD.

9. Menores de Idade

A Plataforma não é destinada a menores de 18 anos. Não coletamos intencionalmente dados de menores. Se tomarmos conhecimento de que dados de menor foram coletados, providenciaremos a eliminação imediata.

10. Alterações nesta Política

10.1. Esta Política pode ser atualizada periodicamente.

10.2. Alterações significativas serão comunicadas pela Plataforma.

10.3. Recomendamos a revisão periódica desta página.
